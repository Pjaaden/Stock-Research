# 现金流可视化规范 — 研究报告标准组件

> 适用于所有公司研究报告，作为"现金流章节"的标准可视化输出。  
> 本规范由报喜鸟2025H1研究中首次建立（2026-04-16）。

---

## 一、适用场景

每份公司深度研究报告，在完成财报分析后，**必须包含一张现金流全流向图**。  
目标是让读者一眼看清：钱从哪来、钱到哪去、期末还剩多少、哪些是"真实消耗"哪些是"科目搬家"。

---

## 二、数据提取清单（提取顺序）

从合并现金流量表逐项提取，精确到亿元（保留2位小数）：

### 经营活动流入
- 销售商品、提供劳务收到的现金（核心，直接反映回款能力）
- 收到的税费返还
- 收到其他与经营活动有关的现金（政府补贴、代理商风险金等）

### 经营活动流出
- 购买商品、接受劳务支付的现金（采购）
- 支付给职工以及为职工支付的现金（薪酬）
- 支付的各项税费
- 支付其他与经营活动有关的现金（销售费用、管理费用的现金部分）

### 投资活动流入
- 收回投资收到的现金（理财赎回为主）
- 取得投资收益收到的现金（理财利息、联营企业分红）
- 处置固定资产、无形资产收回的现金净额（房产出售等）
- 收到其他与投资活动有关的现金

### 投资活动流出
- 购建固定资产、无形资产和其他长期资产支付的现金（毛投资）
- 投资支付的现金（购买理财 + 股权投资）
- 取得子公司及其他营业单位支付的现金净额（收购）
- 支付其他与投资活动有关的现金

### 筹资活动流入
- 吸收投资收到的现金（增发/少数股东出资）
- 取得借款收到的现金（新增贷款）
- 收到其他与筹资活动有关的现金

### 筹资活动流出
- 偿还债务支付的现金（还贷）
- 分配股利、利润或偿付利息支付的现金（分红+利息）
- 支付其他与筹资活动有关的现金

### 期初/期末
- 现金及现金等价物期初余额
- 现金及现金等价物期末余额
- 汇率变动影响（外币敞口大的公司注意）

## 三、关键分析注释（必须在图旁说明）

### 3.1 理财"科目搬家"识别
若投资流入含"赎回理财"、流出含"购买理财"，需注明：  
`净增理财 = 购入 - 赎回`  
如净增为正，则该金额是从货币资金搬到交易性金融资产，**现金未真实消耗**。

### 3.2 投资支付的拆分
"投资支付的现金"常混合理财购买和战略投资（收购股权），需从附注区分：  
- 理财部分 → 科目搬家，非消耗  
- 战略收购部分 → 真实资本支出

### 3.3 经营现金流质量判断
计算：经营现金流 / 归母净利润  
- \>1.0：现金质量良好，利润含金量高  
- 0.5-1.0：一般，存在应收账款占压  
- \<0.5：警示，利润与现金存在明显背离

### 3.4 广义现金计算
广义现金 = 货币资金 + 交易性金融资产（理财）+ 其他债权投资（大额存单）+ 其他非流动金融资产（长期理财/基金）  
减去受限资金（保证金、银行监管资金，详见资产权利受限披露）

## 四、SVG图规范

### 布局结构（从上到下）
```
[期初现金]
    ↓
[① 经营活动区块] — 左列:流入项 / 右列:流出项 / 最右:净额汇总框
    ↓
[② 投资活动区块] — 同上布局
    ↓
[③ 筹资活动区块] — 同上布局
    ↓
[汇总栏] — 三个净额格 + 总净变动格
    ↓
[期末现金]
```

### 颜色规范（使用系统CSS变量色板）
- 流入项：`c-teal`（绿色系，代表钱进来）
- 流出项：`c-coral`（橙红系，代表钱出去）
- 净额正值：`c-teal`
- 净额负值：`c-coral`
- 期初/期末：`c-gray`（中性）
- 分隔线/区块边框：`stroke-dasharray="4 3"`虚线，`var(--color-border-tertiary)`

### 字体规范
- 项目名称：`class="th"`（14px medium）
- 金额：`class="ts"`（12px）放在项目名下方同框
- 区块标签：`class="ts"` + `fill="var(--color-text-secondary)"`

### viewBox
- 宽度固定 680px
- 高度根据行数动态计算，每个活动区块约180-200px，留60px给汇总和标题
- 典型三大活动高度约860px

### 注释
- 若有特殊项目（理财搬家、大额收购），在图底部用 `class="ts"` 加注释行
- 格式：`注①：...`

## 五、Prompt模板（每次生成时使用）

```
从[公司]的[报告期]合并现金流量表提取全部流入/流出数据，
生成标准现金流全流向SVG图，遵循以下规范：
1. 布局：期初→经营区块→投资区块→筹资区块→汇总栏→期末，全部垂直排列
2. 颜色：流入用c-teal，流出用c-coral，净额按正负分别用c-teal/c-coral
3. 投资活动中若有理财购买和赎回，在图底注释净增理财金额及"科目搬家"说明
4. 汇总栏列出三大活动净额和总净变动
5. 图底补充：广义现金 = 货币资金 + 理财 + 其他债权投资，以及受限资金说明
6. viewBox 680×H，H根据内容计算，不留多余空白
```
