## 研究方法论

**估值框架（各类资产）**：
- 商业地产（国贸）：双法交叉验证（PS法营收倍数 + PE法EPS倍数），净现金/股加回
- 轻资产物业管理（华润/绿城）：EV/EBITDA+净现金拆分 + PE + 股息率；ROE因结构性失真不作主参考
- 制造业出海（赛轮/中策）：海外工厂净利润基本盘量化 + 真实资金缺口重构 + 情景概率加权
- 消费成长（霸王茶姬）：GMV增速 + Non-GAAP PE + 净现金占比

**数据标注规则**：★ = 报表原文数字；其余为推算/估算值，明确标注来源。
