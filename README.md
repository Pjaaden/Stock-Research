## 研究方法论说明

**估值框架**（适用于各公司）：
- 轻资产管理公司（华润万象）：EV/EBITDA + 净现金拆分法 + PE + 股息率；PB和ROE因结构性失真不作主参考
- 重资产商业地产（中国国贸）：成本法账面价值 vs 公允价值双法；FCF覆盖倍数
- 制造业（赛轮/中策）：真实资金缺口重构；利息覆盖率；FCF；净负债/EBITDA

**数据标注规则**：★ 为原始报表数据；其余为计算/推算值。
