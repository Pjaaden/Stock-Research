NOI Margin 的财务含义
NOI = Net Operating Income · 净营业收入
计算逻辑： 租金收入（固定租金 + 流水抽成） − 物业直接运营成本（保洁、安保、维修、能耗、人工等） − 空置损失 ────────────────────────────────── = NOI（净营业收入） NOI Margin = NOI ÷ 租金收入

NOI衡量的是一个购物中心在扣除运营成本、但不扣折旧、不扣融资利息、不扣总部分摊之后，纯粹靠"运营这个项目"能赚多少钱。它是评估商业地产单项目运营效率最核心的指标，排除了资本结构和会计处理的干扰。

华润万象2025H1 NOI Margin达到68.2%——每收100元租金，扣掉运营成本后还剩68.2元。这一水平在行业内属于顶尖，反映其高端定位带来的强议价权和低空置率。

折旧摊销：对利润表与现金流的不同影响
折旧是非现金支出——利润表扣了，现金没出去
利润表（折旧压低净利润）： 营业收入 − 直接成本（人工、运营） − 折旧摊销 ← 扣减，影响净利润 = 经营利润（EBIT） − 利息支出 = 税前利润 → 净利润
现金流表（折旧加回）： 净利润 + 折旧摊销加回 ← 非现金，加回 + 使用权资产摊销加回 ← 同上 ± 营运资本变动 ────────────────── = 经营活动现金流
实例：华润万象2025H1经营现金流覆盖核心净利润121.4%——超额的21.4pt，一部分正是折旧加回贡献。对重资产公司：净利润率被折旧"人为压低"，EBITDA和FCF才反映真实盈利能力。
正确衡量指标含义适用场景
经营现金流（CFO）主业造血能力判断主业健康度
自由现金流（FCF）扣资本支出后真实盈余判断分红可持续性
EBITDA剔除折旧后运营利润重资产公司跨期对比
EBITDA利息覆盖倍数真实偿债安全垫信用风险判断
净利润率（慎用）被折旧扭曲重资产公司单独使用意义有限
PB（慎用）轻资产公司账面净资产失真华润万象基本失效

Gordon Growth Model（永续增长模型）详解
第一步：公司值多少钱？从折现开始

公司的价值 = 它未来能给你带来的所有现金，折算成今天的价值。今天的100元比五年后的100元更值钱，这个折算过程叫折现，折算用的利率叫折现率（本质是你的机会成本）。

零增长情况（最简单）：每年稳定赚35亿，折现率10% 第1年：35亿 ÷ 1.10 = 31.8亿 第2年：35亿 ÷ 1.10² = 28.9亿 第3年：35亿 ÷ 1.10³ = 26.3亿 …… 无穷加总 …… 公司价值 = 35亿 ÷ 10% = 350亿 对应PE = 10x → 零增长公司的合理PE = 1 ÷ 折现率
加入增长率g后（Gordon Model核心公式）： 公司价值 = 利润 ÷ (折现率 − 增长率) = 利润 ÷ (r − g) 增长率对估值的指数级影响（折现率10%，利润35亿）： g = 0% → 价值 350亿 PE = 10x g = 5% → 价值 700亿 PE = 20x g = 8% → 价值 1750亿 PE = 50x g = 9% → 价值 3500亿 PE = 100x 增长率从0%→8%，估值涨5倍——增长率微小变化对估值影响是指数级的
第二步：加入ROE——高ROE公司为何值更高PE
增长率 g = ROE × 再投资率（留存再投资的利润比例） 同样实现8%增长，ROE不同，股东拿到的现金天差地别： 公司A（ROE=30%，华润万象类）： 再投资率 = 8% ÷ 30% = 26.7% 赚100元 → 留26.7元 → 分给股东73.3元 公司B（ROE=10%，普通物管）： 再投资率 = 8% ÷ 10% = 80% 赚100元 → 留80元 → 分给股东仅20元
完整Gordon Model公式： 合理PE = (1 − g/ROE) ÷ (r − g) 华润万象（ROE=30%，g=8%，r=10%）： 合理PE = (1 − 8%/30%) ÷ (10% − 8%) = 73.3% ÷ 2% = 36.6x 普通物管（ROE=12%，g=5%，r=10%）： 合理PE = (1 − 5%/12%) ÷ (10% − 5%) = 58.3% ÷ 5% = 11.7x 当前22x PE在理论框架下偏保守——前提是ROE和增长率假设成立 增长率敏感性：g从8%降至6%，合理PE从36.6x腰斩至19.2x

Gordon Growth Model：永续增长估值模型详解
第一步：最基本的问题——一家公司值多少钱

答案只有一个：它未来能给你带来的所有现金，折算成今天的价值。需要"折算"是因为今天的100元比五年后的100元更值钱——这个折算过程叫折现，折算用的利率叫折现率（即你的机会成本/要求回报率）。

最简单情况：公司每年稳定赚35亿，永远不增长，折现率10% 第1年：35亿 ÷ 1.10 = 31.8亿 第2年：35亿 ÷ 1.10² = 28.9亿 第3年：35亿 ÷ 1.10³ = 26.3亿 ……一直加到无穷年…… 数学求和 = 35亿 ÷ 10% = 350亿 公式：公司价值 = 利润 ÷ 折现率 对应合理PE = 1 ÷ 10% = 10倍 ← 零增长公司的合理PE
第二步：加入增长——Gordon Model核心公式
假设利润每年增长g%，折现率r，Gordon Model核心公式： 公司价值 = 利润 ÷ (折现率 r − 增长率 g) 合理PE = 1 ÷ (r − g) 增长率对估值的指数级影响（折现率统一10%，利润35亿）： 增长率 0% → 价值 = 35 ÷ (10%−0%) = 350亿 PE = 10x 增长率 5% → 价值 = 35 ÷ (10%−5%) = 700亿 PE = 20x 增长率 8% → 价值 = 35 ÷ (10%−8%) = 1750亿 PE = 50x 增长率 9% → 价值 = 35 ÷ (10%−9%) = 3500亿 PE = 100x 增长率从0%到8%，估值从350亿涨到1750亿——增长率的微小变化影响是指数级的
第三步：加入ROE——为何高ROE公司值更高PE
公司要产生增长，必须留一部分利润再投资。ROE越高，实现同样增长率所需留存的利润越少，能分给股东的越多。
再投资率 = 增长率 g ÷ ROE 公司A（高ROE = 30%），目标增长8%： 再投资率 = 8% ÷ 30% = 26.7% → 赚100元，留26.7元就能实现8%增长 → 剩余 73.3元 可以分给股东 公司B（低ROE = 10%），目标增长8%： 再投资率 = 8% ÷ 10% = 80% → 赚100元，必须留80元才能实现8%增长 → 只有 20元 可以分给股东 两家公司都"增长8%"，股东实际拿到的现金天差地别
完整Gordon Model公式（含ROE）： 合理PE = (1 − 再投资率) ÷ (r − g) = (1 − g/ROE) ÷ (r − g) 华润万象（名义）：ROE 30%，g = 8%，r = 10% 再投资率 = 8% ÷ 30% = 26.7% 合理PE = 73.3% ÷ 2% = 36.6x 普通物管：ROE 12%，g = 5%，r = 10% 再投资率 = 5% ÷ 12% = 41.7% 合理PE = 58.3% ÷ 5% = 11.7x → 当前22x在此框架下看似低估——但ROE存在会计失真（见CH09）

ROE失真：会计幻觉与三种修正方案
ROE在华润万象案例中为何完全不可参考
表观ROE = 净利润 / 净资产 = 35亿 / ~115亿 ≈ 30% 但净资产为什么只有115亿？ → 因为购物中心资产（数百亿）的产权在置地 → 万象生活净资产是"被抽空后的残余" 直营/加盟类比： 你开餐厅，厨房设备、装修、房租全由父亲出钱 你只出人工 → 你的ROE当然极高 但这不代表你的生意模式多优秀 只代表父亲补贴了你 真实经济ROE应把置地资产并表： 分母暴增至数千亿 → ROE可能只有5-8% → 接近普通重资产商业地产水平
结论：Gordon Model里用30% ROE推出的36x合理PE，建立在会计结构性失真上。正确做法是放弃ROE，从现金流出发重建估值。
修正方案一：现金再投资率法
不用ROE间接推算再投资率，直接观察公司实际把多少现金再投入业务。
现金再投资率 = (Capex − 折旧) / 经营现金流 华润万象2024FY估算： 经营现金流： ~42亿 Capex： ~8亿 折旧加回： ~5亿 净新增投入： ~3亿 现金再投资率 = 3亿 ÷ 42亿 ≈ 7% 分红率 = 93% ← 与接近100%派息吻合 现金ROIC = 经营现金流 / 万象自己投入的现金资本 ≈ 42亿 / 200亿 ≈ 21% 隐含增长率 g = 7% × 21% ≈ 1.5% → 揭示了核心问题：万象的增长主要不靠自身再投资驱动 而是靠置地持续供给新项目的"外生增长" 一旦置地收缩，增长来源中断
修正方案二：净现金拆分法（最符合商业模式）
公司价值 = 现有合同永续价值 + 净现金 + 新增合同期权价值 现有合同价值（稳态，折现率8%，存量自然增长3%）： = 35亿核心净利润 ÷ (8% − 3%) = 700亿人民币 ≈ 763亿港元 净现金（安全垫）： = 178亿人民币 ≈ 194亿港元 新增合同期权价值（第三方扩张+城市空间，保守估算）： = ~100-200亿港元（难以精确定价） 合计合理市值 ≈ 763 + 194 + 150 = ~1107亿港元 对应股价 ≈ 48-50港元 当前44港元有小幅安全边际，主要来自对期权价值的低估
关键洞察：置地供给旺盛时，期权价值很大；置地收缩（现正发生）时，期权价值缩水——这正是股价从50跌至44的真实原因。
修正方案三：FCF Yield反推法（最直觉）
跳过所有模型假设，直接问：用44港元买入，每年实际拿回多少现金？
FCF Yield = 自由现金流 / 市值 FCF（2024）= 经营现金流42亿 − Capex8亿 = 34亿人民币 ≈ 37亿港元 FCF Yield = 37亿 ÷ 1005亿 ≈ 3.7% 综合年化回报 = FCF Yield 3.7% + 存量自然增长 3-4% ≈ 7-8% 与机会成本对比： 10年期中国国债： ~2.3% → 万象溢价显著 ✓ 有吸引力 港股高息股平均： ~5-6% → 万象有小幅溢价 ✓ 合理 恒生指数长期回报：~6-7% → 万象基本持平 △ 无超额 私募/实业要求： ~12-15% → 万象明显不足 ✗ 不合适
结论：7-8%综合回报是否值得买入，完全取决于个人机会成本，而非模型给出的"合理PE"。对要求回报率5-6%的投资者有吸引力；对要求10%+的投资者则明显高估。

① 华润万象生活报表只含管理费分成（~6-10%），不含购物中心全部租金——购物中心核心资产收益仍在华润置地。

② 万象生活是"乙方品牌壳"：净资产不含产权，PB失效；真实价值在管理合同现金流、品牌溢价、7200万会员生态，只在PE/FCF Yield/股息率中体现。

③ 折旧是非现金支出，不影响现金流。重资产公司净利润率被折旧压低，EBITDA和FCF才是真实盈利能力的镜子。

④ 母公司持股62%，分红利益完全一致，100%派息率的背后是母公司主动推动超额分配的利益驱动。

⑥ 正确估值公式：期望市值 = EBITDA × 期望倍数 + 净现金。当前44港元对应FCF Yield约3.7%，综合年化回报约7-8%，是否吸引力取决于个人机会成本。

⑦ 置地财务压力是"利润承压"而非"流动性危机"，对万象生活的真实影响是增速天花板（直营新开6座/年），而非业务威胁。置地主动转型轻资产反而是万象的隐性正向驱动。
